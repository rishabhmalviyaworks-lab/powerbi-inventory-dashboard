Why This Project Is Valuable

**About This Project**

This project showcases my ability to analyze large inventory datasets and convert them into actionable business insights using Power BI, DAX, and data modeling. The dashboard provides clear visibility into product stock distribution, seller performance, category trends, and customer behavior metrics.

 **Why This Project Is Valuable**

1. Demonstrates advanced Power BI reports for retail/inventory analysis

2. Highlights skills in data cleaning, modeling, and analytical storytelling

3. Reflects a real-world business use case relevant to data analyst & BI analyst roles

4. Shows ability to derive insights that support strategic decisions

**Key Features**

1. Category-wise, brand-wise & gender-wise stock analysis

2. Seller performance comparison

3. State-wise stock distribution map

4. Conversion funnel: Views → Add to Cart → Orders

5. Average rating and product quality insights

6. Fully interactive slicers and drill-down views

** Tools Used**

1. Power BI Desktop

2. Pwer Query (M Language)

3. DAX

4. Excel / CSV Dataset

 **Skills Demonstrated**

1. Data Cleaning & Transformation

2. Data Modeling (Star Schema)

3. KPI Development using DAX

4. Dashboard Design & Visualization

5. Insight Generation & Storytelling

 **Key Outcomes**

1. Improved visibility into stock patterns and demand segments

2. dentified overstock and understock categories

3. Enabled data-driven inventory and procurement planning

4. Highlighted customer behavior and conversion gaps
